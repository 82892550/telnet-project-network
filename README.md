# telnet-project-network
Telnet Network Project
📌 Overview

This project demonstrates how to configure and use the Telnet protocol to remotely access network devices such as routers and switches using Cisco Packet Tracer.

🎯 Objectives

Configure Telnet on a router

Enable remote access from a PC

Understand basic network communication

Practice IP addressing and routing

🛠️ Tools & Technologies

Cisco Packet Tracer

Telnet Protocol

Networking Basics (IP Addressing, Routing)

🧩 Network Topology

The network consists of:

Router

Switch

End Devices (PCs)

Each device is configured with appropriate IP addresses to allow communication.

⚙️ Configuration Steps
1. Assign IP Address to Router
2. Configure Telnet Access
3. Configure PC

Assign IP address (e.g., 192.168.1.2)

Set default gateway (192.168.1.1)

4. Test Connection

From PC command prompt:

✅ Expected Result

Successful remote access to the router via Telnet

Ability to execute commands on the router remotely

⚠️ Notes

Telnet is not secure (data is sent in plain text)

It is recommended to use SSH instead in real-world networks
